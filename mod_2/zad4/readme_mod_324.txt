Не большое усложнение программы 1. 
Добавлена возможность проверки в начале работы программы наличия файла с таким именем в текущей директории.
Две программы. Обе для запуска требуют имя файла в параметрах запуска. Используется небуферизированный ввод вывод.
Вторая программа считывает данные из файла в динамический массив который увеличится до нужного объема автоматически.
