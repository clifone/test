Программа имеет неопределеное количество входных параметров. Предпологается что парaметрами будут являться числа, причем их число не должно быть меньше двух. В коде есть соответствующие проверки этого.

Выход:
По итогу работы выводиться строки для родительского процесса и для дочернего. Какждый процесс вычисляет площади квадратов по параметрам. Число вычислений делиться примерно по ровну между процессами.
