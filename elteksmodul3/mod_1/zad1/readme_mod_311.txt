Программа не требует в ходных данных.

Выход:
По итогу работы выводиться шесть строк:
1 PID текущего процеса
2 PPID текущего процеса
3 UID текущего процеса
4 EUID текущего процеса
5 GID текущего процеса
6 SID текущего процеса для этого в качестве входного параметра функции geteuid() ставиться 0

Программа проверяет работу системных вызовов get*
