Программа имеет неопределеное количество входных параметров которые ни на что не влияют

Выход:
По итогу работы выводиться строки для родительского процесса и для дочернего. Выводятся сами информация о том какой это процесс и входные параметры. После чего при успешном завершении, выполняется функция, печатающая строку об успешном завершении и значение которое вернул вызов функции fork() (0 для дочернего, > 0 для родителя).
